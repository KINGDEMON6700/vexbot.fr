# audit-coherence

Fais un audit visuel et textuel du panel Vex.
Dis-moi :
1. Les éléments UI qui ne sont pas cohérents entre les pages
2. Les textes qui semblent trop techniques ou générés par IA
3. Les composants similaires qui ont été créés en double
4. Ce qui manque en termes d'aides contextuelles pour l'utilisateur
Ne touche à rien, donne-moi juste le rapport.

This command will be available in chat with /audit-coherence
